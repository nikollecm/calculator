# 💭 Por Que Construí Isso

Este projeto foi criado como uma forma prática de aprender e implementar os fundamentos do Flutter e gerenciamento de estado. A ideia era construir algo funcional e útil enquanto explorava conceitos essenciais como:

- **Gerenciamento de estado local** - Como manter e atualizar dados reativamente
- **Arquitetura limpa** - Separação de responsabilidades (UI, lógica, dados)
- **Widgets customizados** - Criação de componentes reutilizáveis
- **Lógica de negócio** - Implementação de operações matemáticas e validações
- **Design responsivo** - Interface adaptável e intuitiva

Escolhi um app de calculadora porque é simples o suficiente para focar no aprendizado da estrutura do Flutter, mas completo o suficiente para cobrir conceitos importantes como manipulação de eventos, formatação de dados e organização de código em camadas.

## 🎯 Objetivos de Aprendizado

### 1. Dominar o Básico do Flutter

Entender como widgets funcionam, como compor interfaces e como o Flutter renderiza a UI de forma eficiente.

### 2. Implementar Lógica de Negócio

Separar a lógica de cálculo da interface, criando um código testável e mantível através da camada de serviços.

### 3. Organizar Código Profissionalmente

Estruturar o projeto em pastas lógicas (constants, models, services, screens, widgets) seguindo boas práticas da indústria.

### 4. Trabalhar com Estado

Gerenciar estado de forma eficiente usando StatefulWidget e entender quando e como atualizar a UI.

## 🚀 O Que Aprendi

### Gerenciamento de Estado

Cada operação na calculadora envolve mudanças de estado - números digitados, operações selecionadas, resultados calculados. Aprendi a controlar esse fluxo de dados de forma previsível.

### Arquitetura em Camadas

Separar a UI (widgets) da lógica (services) e dos dados (models) tornou o código muito mais organizado e fácil de manter.

### Tratamento de Edge Cases

Lidar com divisão por zero, limites de display, formatação de decimais e encadeamento de operações me ensinou a pensar em todos os cenários possíveis.

### Design de Interface

Criar uma interface limpa e intuitiva, escolhendo cores harmoniosas e garantindo que os botões sejam fáceis de usar.

## 💡 Desafios Superados

- **Encadeamento de operações**: Fazer 5 + 3 + 2 funcionar corretamente exigiu pensar cuidadosamente sobre quando calcular resultados intermediários
- **Formatação de números**: Decidir quando mostrar decimais e quando mostrar inteiros
- **Organização do código**: Refatorar de um arquivo único para uma estrutura modular e escalável

## 🎨 Por Que Este Design?

Escolhi um tema rosa suave para criar uma experiência visual agradável e diferente das calculadoras tradicionais. A fonte Jersey 15 adiciona personalidade sem comprometer a legibilidade.
